# 🛠️ Validador de CPF em Python.

Projeto desenvolvido para praticar lógica de programação utilizando Python.

Este projeto foi desenvolvido para aprimorar aprimorar o raciocínio lógico utilizando Python, aperfeiçoar estruturas importantes como (laços, if/else) e também trabalhar conceitos importantes da linguagem.

## 🚀 Funcionalidades do projeto
- Entrada de CPF
- Tratamento e limpeza dos dados de entrada
- Validação dos dígitos verificadores do CPF
- Menu interativo
  
## 📚 Conceitos aplicados
- Tratamento e limpeza da entrada
- Estrutura de repetição (for)
- Estruturas condicionais
  
## 💾 Tecnologias utilizadas
- Python 3
